# v1.0.3 릴리즈 노트 초안

`v1.0.2` 이후 입력 검증과 메인 설정 UX를 다듬은 패치 릴리즈 초안입니다.

## 핵심 변경
- Windows 자동 실행 토글을 고급 설정에서 메인 작업 설정 영역으로 이동
- 문서 대상 잠금을 수동 잠금과 감시 중 자동 잠금으로 분리
- Google Docs URL 또는 문서 ID 입력 검증을 강화해 잘못된 문서 형식을 빠르게 차단
- 정규식 필터 편집과 저장 흐름에 유효성 검사와 테스트 결과 피드백을 추가
- 수동 Google 재인증을 브라우저 승인 중심 흐름으로 명확히 분리

## 품질
- 메인 UI 구조, 문서 대상 잠금, 입력 검증, 고급 필터 적용 흐름 테스트 확장
- 릴리즈 버전 메타데이터 정합성 테스트 유지

## 릴리즈 메타데이터
- 패키지 버전: `1.0.3`
- 릴리즈 워크플로 기본 태그: `v1.0.3`
