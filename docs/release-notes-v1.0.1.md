# v1.0.1 릴리즈 노트 초안

`v1.0.0` 이후 메인 워크스페이스 UI와 감시 시작 흐름을 중심으로 정리한 초안입니다.

## 주요 변경

- 메인 창 하단에 대형 `감시 시작` CTA를 고정 배치하고 준비도 게이트를 추가했습니다.
- `작업 로그`와 `최근 추출 결과`를 하나의 활동 탭으로 통합했습니다.
- 최근 추출 결과를 카드형 요약으로 재구성하고 최근 3건 중심으로 표시합니다.
- 저장 안 됨 배지, 현재 처리 파일, 마지막 성공/결과 요약을 상단 상태 패널에 추가했습니다.
- 기본 설정과 접히는 고급 설정을 분리해 반복 사용 시 세로 공간 활용을 개선했습니다.

## 사용자 영향

- 감시를 시작하기 전에 어떤 입력이 부족한지 바로 확인할 수 있습니다.
- 오류가 발생하면 로그 탭으로 자동 전환되어 원인 파악이 빨라집니다.
- 자주 쓰는 실행 흐름은 단순해지고, 고급 옵션은 필요할 때만 펼쳐 사용할 수 있습니다.

## 배포 확인 항목

- `pytest -q` 통과
- 릴리즈 워크플로 기본 태그 `v1.0.1` 반영
- 패키지 버전과 내부 버전 문자열 일치
