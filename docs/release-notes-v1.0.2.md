# v1.0.2 릴리즈 노트 초안

`v1.0.1` 이후 Google 인증 UX와 정책을 재설계한 변경을 정리한 초안입니다.

## 핵심 변경
- 대화형 Google 재인증을 foreground 사용자 승인 흐름으로 분리
- 백그라운드 경로에서 브라우저 인증 자동 실행 차단
- 토큰 불일치/갱신 실패 시 `token.invalid.YYYYMMDD-HHMMSS.json` 격리 정책 추가
- 감시 시작/문서 액션의 Google 서비스 준비 로직 공통화
- 보조 액션에 `Google 계정 다시 연결`, `인증 초기화` 버튼 추가

## 품질
- 인증 정책/백그라운드 차단/재인증 분기 관련 테스트 확장
- 전체 테스트 통과 확인

## 릴리즈 메타데이터
- 패키지 버전: `1.0.2`
- 릴리즈 워크플로 기본 태그: `v1.0.2`
