# v1.0.0 릴리즈 노트 초안

`v0.6.1` 이후 사용자 체감 변경을 기준으로 정리한 초안입니다.

## 주요 변경

- 시작 가이드 대화상자의 문구와 구성 가독성을 개선했습니다.
- Windows 시작 프로그램 설정 흐름을 더 명확하게 다듬었습니다.
- GitHub 공식 릴리즈 기준의 새 버전 확인 기능을 추가했습니다.
- 작업 결과 팝업, 트레이 알림, 최근 추출 결과 표시 흐름을 정리했습니다.
- 패키징 메타데이터와 Windows 배포 스크립트를 정식 릴리즈 기준으로 정비했습니다.

## 사용자 영향

- 처음 실행하는 사용자가 문서 연결 흐름을 더 쉽게 이해할 수 있습니다.
- 앱 시작 시 최신 공식 배포 버전이 있는지 바로 확인할 수 있습니다.
- 배포물은 폴더형 zip과 단일 실행 파일 형태를 모두 제공합니다.

## 배포 확인 항목

- `token.json` 미포함
- 사용자별 설정 파일, 로그, 캐시 미포함
- `developer_credentials.json.example` 포함
- Windows PowerShell 기준 빌드 성공
